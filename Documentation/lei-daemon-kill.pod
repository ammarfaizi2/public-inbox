=head1 NAME

lei-daemon-kill - signal the lei-daemon

=head1 SYNOPSIS

lei daemon-kill [-SIGNAL | -s SIGNAL | --signal SIGNAL]

=head1 DESCRIPTION

Send a signal to the lei-daemon.  C<SIGNAL> defaults to C<TERM>.
