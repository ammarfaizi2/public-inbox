=head1 NAME

lei-mail-diff - diff the contents of emails

=head1 SYNOPSIS


lei mail-diff [OPTIONS] LOCATION

lei mail-diff [OPTIONS] (--stdin|-)

=head1 DESCRIPTION

This is a diagnostic command that's useful for finding deduplication
bugs.
