=head1 NAME

lei-ls-label - list labels

=head1 SYNOPSIS

lei ls-label [OPTIONS]

=head1 DESCRIPTION

List all known message labels ("mailboxes" in JMAP terminology).

=head1 OPTIONS

=over

=item -z, -0

Use C<\0> (NUL) instead of newline (CR) to delimit lines.

=item -q, --quiet

Suppress feedback messages.

=back
